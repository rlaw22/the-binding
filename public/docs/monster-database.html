Usage in The Binding

ModeHow Monsters Are Used
Campaign ModeAI DM selects monsters by environment, theme, and story context. Encounters balanced to party level using DMG XP thresholds.
Digital DM ModeFull access to all 334 SRD creatures. Can create custom encounters. Legendary monsters have legendary + lair actions.
Storyline ModeAdventures reference specific monsters by name. Stat blocks pulled during combat scenes from this database.

Future: Custom Monsters

The Binding will extend beyond the SRD with original creatures in data/monsters/binding-custom.json using the same schema. The Monster Manual module will automatically merge both databases.
