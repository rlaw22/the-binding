## How to Use

Each adventure has a **Level Range** that indicates what character levels it's designed for.
Players will see this in the adventure selection dropdown so they can pick content appropriate for their character.

- **Fixed**: Adventure is designed for a specific level (e.g. "Level 4")
- **Range**: Adventure spans multiple levels (e.g. "Levels 1–5")
- **All Levels**: Adventure scales to any character level

## Notes

- All adventures need to be converted to The Binding's scene graph format before they're playable
- Level ranges are based on the original adventure design; The Binding's rubber-band difficulty scaling may allow wider ranges
- The dropdown in the UI should show: `"Adventure Name (Levels X–Y)"` or `"Adventure Name (Level X)"`
- Adventures marked as "Free" have publicly available PDFs/content that can be downloaded and converted
